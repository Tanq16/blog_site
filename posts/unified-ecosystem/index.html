Bookmark Manager

  • Notion
  • Booky
  • Raindrop.io
  • Shiori

Note Taking (mainly for computer science folks)

  • Notion
  • Obsidian
  • GitHub

Instant Messaging

  • Telegram
  • WhatsApp
  • Signal

Social Group/Team Chat + Collaboration

  • Discord
  • Slack
  • Mattermost
  • Fleep
  • Microsoft Teams

Password Manager

  • Dashlane
  • Bitwarden
  • 1Password

VPN

  • Surfshark
  • NordVPN
  • ProtonVPN
  • Mozilla VPN
  • Psiphon (fully free)
  • Windscribe

Finance Tracking

  • Spendee
  • BudgE Docker Image by LSIO
  • NerdWallet
  • YNAB

  • Notion

Web News Consolidator

  • Google News
  • Feedly
  • Flipboard

Music Streaming

  • Spotify
  • Apple Music

  • Amazon Music

Cloud Storage

  • Google Drive
  • Box
  • Mega Drive
  • Icedrive
  • DigiBoxx (Specially for India)

Reminders and To-Do lists

  • TickTick
  • Todoist
  • Notion
  • Microsoft ToDo
  • Google Calendar

Project Management and Collaboration

  • Notion
  • Airtable
  • ClickUp
  • Asana

Email + Calendar (Not Paid Office Suits)

  • ProtonMail
  • Microsoft Outlook
  • Gmail by Google

Email Aggregator Application

  • Spark
  • Outlook
  • Missive

Private Email Address System

  • SimpleLogin
  • Firefox Relay
  • AnonAddy

Money Transfers

  • Wise (Formerly TransferWise)
  • Xoom
  • Remitly

Browser

  • Google Chrome
  • Brave Browser
  • Firefox
  • Vivaldi Browser
  • Microsoft Edge

Social Platform

  • Twitter
  • Reddit
  • LinkedIn
  • Instagram
  • Facebook

Authenticator App

  • Authy
  • Microsoft Authenticator App
  • LastPass Authenticator
  • Google Authenticator

AdBlockers

  • Adblock Plus
  • AdGuard
  • Brave Browser

File Sharing

  • SnapDrop
